# Apps

This directory contains the main applications for the Eden Music Scene:

## Structure

- **backend/**: Python-based services and tools
  - `EchoDJ/` - DJ/curation agent
  - `EchoPlay/` - Playback & ritual listening client
  - `EdenOS_EchoShare/` - Shared libraries
  - `music_files/` - Music catalog and metadata
  - `emotion_*.py` - Emotion tagging services

- **frontend/**: TypeScript/React web application (EchoSplit)
  - Web-based creation studio
  - React + Vite + TypeScript
